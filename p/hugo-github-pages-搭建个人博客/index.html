手动搭建博客经历

手动方式：

  1. 安装：hugo, go.
  2. clone 主题：git submodule add theme.
  3. hugo server 本地调试。
  4. 设置GitHub Pages主页信息
  5. 使用GitHub Actions 自动构建/部署（CI/CD）
  6. 本地提交到GitHub仓库

上述过程可以参考 通过 Github workflows CI/CD 自动化部署 Github Pages hugo 免费博客
