4.1жҹҘзңӢжҳҜеҗҰжҸҗдәӨжҲҗеҠҹ

иғҢжҷҜпјҡеңЁwwh_devеҲҶж”ҜдёҠејҖеҸ‘пјҢжҸҗдәӨд»Јз ҒеҗҺпјҢmergeд№ӢеҗҺпјҢжғіеңЁе…¶д»–еҲҶж”ҜзңӢжҲ‘жҸҗдәӨзҡ„еҶ…е®№пјҢеҗҢж—¶дҝқиҜҒдёҖдёӘеҲҶж”Ҝж—¶еҲ»дҝқжҢҒжңҖж–°гҖӮй—®йўҳжқҘдәҶгҖӮеңЁThermal5.0пјҲжғіиҰҒдҝқжҢҒжңҖж–°зҡ„еҲҶж”ҜпјүзңӢдёҚеҲ°жҲ‘жҸҗдәӨзҡ„еҶ…е®№гҖӮ

еҺҹеӣ еңЁдәҺжІЎжңүиҝӣиЎҢеҲҶж”ҜиҝҪиёӘпјҡ

    2. вҖӢdevвҖӢеҲҶж”ҜпјҢжҜҸж¬ЎжңҖж–°д»»еҠЎзҡ„ејҖеҸ‘еҲҶж”Ҝ
    3. вҖӢdev_xx_backupвҖӢеҲҶж”ҜпјҢејҖеҸ‘е®ҢдёҖдёӘд»»еҠЎеҗҺзҡ„еӨҮд»ҪпјҢжӢ·иҙқжқҘиҮӘdevвҖӢеҲҶж”ҜпјҢеҢ…еҗ«д»»еҠЎжөӢиҜ•д»Јз Ғ

  2. еҲҶж”Ҝз®ЎзҗҶ

    1. жҜҸж¬ЎpushвҖӢжҸҗдәӨеҗҺпјҢзӯүеҫ…з®ЎзҗҶе‘ҳжЈҖжҹҘж— иҜҜеҗҺпјҢе°ҶdevвҖӢеҲҶж”ҜеҲ йҷӨ
    2. дҝқиҜҒmainвҖӢеҲҶж”ҜжңҖж–°пјҢд»ҺmainвҖӢеҲҶж”Ҝдёӯж–°е»әдёҖдёӘеҲҶж”ҜеҪ“жҲҗеҪ“еүҚд»»еҠЎзҡ„еҲҶж”Ҝ
    3. жӯӨж—¶branchвҖӢз»“жһ„дёӯеҸӘеҢ…еҗ«пјҢдёҖдёӘmainвҖӢеҲҶж”ҜпјҢдёҖдёӘdevвҖӢеҲҶж”ҜпјҢе’ҢжҜҸж¬Ўд»»еҠЎзҡ„еӨҮд»ҪвҖӢеҲҶж”Ҝ
